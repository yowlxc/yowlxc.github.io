# Лабораторная работа №2

## Цель

- Изучить и освоить возможности библиотек Matplotlib, NumPy и Pandas в Python для анализа и визуализации данных

## Выполненные задания

1. Изучены и реализованы операции с массивами (создание, reshape), векторами (создание, сложение, умножение на число, поэлементное умножение, скалярное произведение) и матрицами (создание, умножение, нахождение определителя, нахождение обратной матрицы, решение уравнения вида Ax = b в матрицах)
2. Изучены операции статического анализа: загрузка csv-файла и преобразование его данных в массив, статистический анализ (среднее, медиана, стандартное отклонение, минимум, максимум, 25 и 75 перцентили), min-max нормализация.
3. Освоены методы визуализации данных с помощью Matplotlib: построение гистограммы, тепловой карты, графика зависимости x -> y
4. Проведены рекомендуемые тесты, все тесты проходят проверку
